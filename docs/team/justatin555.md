## Overview: B2B4U

B2B4U is a desktop contact management application made for business consultants, optimized for use via a Command Line Interface (CLI) while still having the benefits of a Graphical User Interface (GUI).
If you can type fast, B2B4U can get your human resource management tasks done faster than traditional GUI apps.

Given below are my contributions to the project.

## New features
- Added the ability to sort contacts by various fields
  - Function: Allow users to sort their contacts by various fields such as name, company, last contacted date and ranked tags.
  - Justification: This feature allows users to easily find the contact they are looking for, and also allows them to easily view their contacts in a way that suits their needs.
  - Credits: Builds on Xing Yu's interface for sorting.
  - Non-trivial pull-requests: [#121](https://github.com/AY2526S2-CS2103T-T08-1/tp/pull/121)
- Added the ability to delete notes
  - Function: Allow users to remove unnecessary or outdated notes from their contacts.
  - Justification: This feature allows users to keep their contact information up-to-date and relevant.
  - Credits: Builds on @deltaMinor's implementation of the note feature.
  - Non-trivial pull-requests: [#85](https://github.com/AY2526S2-CS2103T-T08-1/tp/pull/85), [#184](https://github.com/AY2526S2-CS2103T-T08-1/tp/pull/184)

### Enhancements to existing features
- Added ranked tags
  - Function: Allow users to define a custom sort order for their users.
  - Justification: This feature allows users to have more control over how their contacts are sorted, and allows them to easily view their contacts in a way that suits their needs.
  - Non-trivial pull-requests: [#111](https://github.com/AY2526S2-CS2103T-T08-1/tp/pull/111)
